# Frontend demo

Ten moduł dostarcza React SPA z produktami, formularzem zamówienia i stroną security demo.

## Wymagania

- Node.js 22+
- npm 10+

## Komendy

```powershell
cd frontend
npm install
npm test
npm run build
npm run dev
```

## Strony

- `/products` — lista produktów z backendu Java
- `/order` — formularz zamówienia i mapowanie błędów walidacji
- `/security-demo` — przykłady podatności po stronie przeglądarki
- `/login` — placeholder dla redirectu 401
