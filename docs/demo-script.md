# Demo script

1. Show the architecture: React SPA on the left, Java backend on the right.
2. Explain that secure Java does not automatically secure browser-side JavaScript.
3. Open `/products` and show the backend-backed catalog.
4. Open `/order` and submit a valid order.
5. Show the validation errors on the form and explain the `fieldErrors` contract.
6. Navigate to `/security-demo` and compare frontend risks with backend Java risks.
7. Open the GHAS workflows and show where CodeQL, Dependabot, and Dependency Review would surface findings.
