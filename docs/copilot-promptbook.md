# Copilot promptbook

## Context

This repository demonstrates how GitHub Copilot can help build a React SPA, a Java backend, and a GHAS story around them.

## Main prompt

"Refactor the ProductCard component so it supports keyboard navigation, accessibility labels, and a cleaner layout while keeping the backend contract unchanged."

## Expected outcome

- A11y-friendly product card
- Keyboard interaction for Enter and Space
- Focusable and screen-reader friendly UI

## Shorter prompt variant

"Make ProductCard accessible and keyboard friendly."

## More detailed prompt variant

"Review the products page and improve ProductCard, useProducts, and the order form for accessibility, error handling, and testability."

## What to say to the customer

"Copilot is useful not only for generating code, but also for making the demo code safer and easier to explain."
